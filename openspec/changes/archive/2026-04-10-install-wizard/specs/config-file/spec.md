## ADDED Requirements

### Requirement: MetisConfig struct and YAML serialization
The system SHALL define a `MetisConfig` struct in `internal/config/config.go` with fields: `db_driver` (string), `db_dsn` (string), `jwt_secret` (string), `license_key_secret` (string). The struct SHALL serialize to/from YAML format.

#### Scenario: Serialize config to YAML
- **WHEN** a MetisConfig is saved to disk
- **THEN** the output file SHALL be valid YAML with a header comment "# Metis Configuration — auto-generated by installer"

#### Scenario: Deserialize config from YAML
- **WHEN** a metis.yaml file is read
- **THEN** the system SHALL parse all four fields into a MetisConfig struct

### Requirement: Load config file
The system SHALL provide a `Load(path string)` function that reads and parses metis.yaml. If the file does not exist, it SHALL return a sentinel error `ErrConfigNotFound`.

#### Scenario: Config file exists
- **WHEN** `Load("metis.yaml")` is called and the file exists
- **THEN** the function SHALL return a populated MetisConfig

#### Scenario: Config file not found
- **WHEN** `Load("metis.yaml")` is called and the file does not exist
- **THEN** the function SHALL return `ErrConfigNotFound`

#### Scenario: Config file malformed
- **WHEN** `Load()` is called with an invalid YAML file
- **THEN** the function SHALL return a parse error

### Requirement: Save config file
The system SHALL provide a `Save(path string)` method that writes the MetisConfig to a YAML file with file permission 0600.

#### Scenario: Write config during installation
- **WHEN** `Save("metis.yaml")` is called during installation
- **THEN** the file SHALL be written with permission 0600 (owner read/write only)

### Requirement: Default SQLite config
The system SHALL provide a `DefaultSQLiteConfig()` function that returns a MetisConfig with `db_driver: "sqlite"` and `db_dsn: "metis.db?_pragma=foreign_keys(1)&_pragma=journal_mode(WAL)"`, with empty secret fields (to be generated during installation).

#### Scenario: Get default config
- **WHEN** `DefaultSQLiteConfig()` is called
- **THEN** the returned config SHALL have db_driver="sqlite" and the standard SQLite DSN with foreign keys and WAL mode

### Requirement: Generate secrets
The system SHALL provide a `GenerateSecrets()` method that fills `jwt_secret` and `license_key_secret` with cryptographically random 64-character hex strings using `crypto/rand`.

#### Scenario: Generate secrets
- **WHEN** `GenerateSecrets()` is called on a MetisConfig
- **THEN** both `jwt_secret` and `license_key_secret` SHALL be set to 64-character hex strings generated from 32 bytes of `crypto/rand`
