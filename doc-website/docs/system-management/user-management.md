# 用户管理

用户管理用于维护 Metis 登录账号及其访问身份。你可以在这里创建、更新、停用和删除用户，并指定用户角色与直属上级。

## 一、功能概览

用户管理用于统一维护系统账号生命周期，确保“谁可以访问系统、拥有哪些角色、当前是否可登录”始终可控。

你可以在该功能中完成账号开通、信息维护、状态控制和账号清理，并将权限模型与组织岗位变化保持同步。

### 前提条件

1. 你拥有用户管理对应权限（查看/新建/编辑/删除）。
2. 系统中已有可分配角色。
3. 你已明确该账号对应的岗位职责。

---

## 二、功能清单

### 功能 1：新建用户

功能介绍：
用于为新成员开通系统访问账号，并在创建时完成角色初始化。

使用场景：
- 新员工入职
- 外部协作人员短期接入
- 项目上线前集中开通账号

### 功能 2：编辑用户

功能介绍：
用于维护用户基础资料与管理关系，包括角色、联系方式、直属上级。

使用场景：
- 员工转岗导致权限变化
- 组织调整导致汇报关系变化
- 联系方式更新

### 功能 3：启用/停用用户

功能介绍：
用于快速控制账号可用状态，不改动账号主体信息。

使用场景：
- 临时冻结账号
- 离岗待交接
- 安全事件临时处置

### 功能 4：删除用户

功能介绍：
用于永久移除不再使用的账号，清理历史账号池。

使用场景：
- 人员长期离职且确认无需保留活跃身份
- 错建账号清理

## 三、常见任务操作

### 任务 1：新建用户

路径：系统管理 > 用户管理 > 新建用户

步骤：
1. 单击“新建用户”。
2. 填写必填项：用户名、密码、角色。
3. 按需填写邮箱、手机号、直属上级。
4. 单击“保存”。

结果：
- 用户立即进入用户列表。
- 用户的菜单可见性与操作权限按角色生效。

影响：
- 影响该用户后续可访问范围。
- 不影响其他用户权限。

可撤销性：
- 可停用。
- 可删除（不可恢复原状态）。

### 任务 2：编辑用户

路径：系统管理 > 用户管理 > 编辑

步骤：
1. 在目标用户行单击“编辑”。
2. 修改邮箱、手机号、角色或直属上级。
3. 单击“保存”。

结果：
- 用户资料即时更新。
- 角色变更后，新权限即时生效。

影响：
- 影响该用户权限边界与管理关系。

可撤销性：
- 可再次编辑回退。

### 任务 3：启用/停用用户

路径：系统管理 > 用户管理 > 启用 / 停用

步骤：
1. 在目标用户行选择“启用”或“停用”。
2. 完成状态切换。

结果：
- 停用后账号不可访问。
- 启用后账号恢复访问。

影响：
- 仅影响账号可用性，不直接修改业务数据。

可撤销性：
- 可反向切换。

### 任务 4：删除用户

路径：系统管理 > 用户管理 > 删除

步骤：
1. 单击“删除”。
2. 在确认框中确认。

结果：
- 用户从列表移除。
- 该身份不再可用于登录与后续分配。

影响：
- 影响后续人员选择范围。

可撤销性：
- 不可撤销。

---

## 四、风险与注意事项

- 删除前先确认该账号是否仍需用于审计回溯。
- 建议优先“停用”，确认后再“删除”。
- 角色分配遵循最小权限原则，不默认给高权限角色。
